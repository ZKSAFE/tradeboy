# TradeBoy - RG34XX交易平台

## 🛒 项目简介

TradeBoy是一个专为RG34XX掌机开发的商品交易平台，提供简洁直观的商品浏览和购买体验。支持多语言显示、手柄操作和自动退出功能。

## 🔧 技术实现

### 核心技术栈
- **SDL2**: 图形渲染和事件处理
- **SDL2_ttf**: 文字渲染和多语言支持
- **Noto Sans CJK**: 中文字体支持
- **Docker**: 跨平台编译环境

### 关键特性
- **UTF-8编码**: 完整的多语言字符支持
- **事件驱动**: 响应式用户交互
- **内存管理**: 安全的资源分配和释放
- **跨平台**: 条件编译支持不同平台
